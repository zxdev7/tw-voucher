## Truewallet Voucher API

This application provides API endpoints for verifying and redeeming Truewallet vouchers.

### API Endpoints

#### 1. Verify Voucher

Verifies a Truewallet voucher without redeeming it.

```
GET /gift-truewallet/verify/:code?mobile=:mobile
```

**Parameters:**
- `code` (path parameter): The voucher code/hash to verify
- `mobile` (query parameter): The mobile number associated with the voucher

**Example Request:**
```
GET /gift-truewallet/verify/ABCD1234?mobile=0812345678
```

**Example Response:**
```json
{
  "status": {
    "code": "SUCCESS",
    "message": "success"
  },
  "data": {
    "voucher": {
      "amount": 100,
      "detail": "Gift voucher",
      "expire_date": "2025-12-31T23:59:59+07:00",
      "link": "https://gift.truemoney.com/campaign/?v=ABCD1234"
    }
  }
}
```

#### 2. Redeem Voucher

Redeems a Truewallet voucher and transfers the funds to the specified mobile number.

```
POST /gift-truewallet/redeem
```

**Request Body:**
```json
{
  "code": "ABCD1234",
  "mobile": "0812345678"
}
```

**Parameters:**
- `code` (string): The voucher code/hash to redeem
- `mobile` (string): The mobile number to receive the funds

**Example Response:**
```json
{
  "status": {
    "code": "SUCCESS",
    "message": "success"
  },
  "data": {
    "voucher": {
      "amount": 100,
      "redeemed_amount": 100,
      "transaction_id": "12345678",
      "redeemed_at": "2025-05-29T14:25:30+07:00"
    }
  }
}
```

**Error Response:**
```json
{
  "statusCode": 400,
  "message": "Voucher not found or already redeemed",
  "error": "Bad Request"
}
```
